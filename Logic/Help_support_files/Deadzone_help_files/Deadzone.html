Deadzone.vi

Help by FRC Team 836 - The RoboBees

This VI allows the user to define a "Deadzone" for whatever numerical parameter is desired. The VI will compare the

input signal to the user defined deadzone.

If the input is in the deadzone a "0" is output.

If outside the deadzone the input signal is passed through.

A front panel indication is given of the comparison result.
